[2025 - now] Waterjet Blade Cutting Robot (MIIT project)

This project, a joint enterprise-university research initiative, utilizes Gmapping for SLAM mapping and GPS+IMU filtering for fusion-based positioning. It also performs 3D terrain mapping via drone LiDAR. Based on robotic arm inverse kinematics control, the project plans blade cutting paths using point cloud data from the host computer, ensuring singularity avoidance. A global 3D reconstruction of a wind turbine model is generated using depth camera SLAM mapping, outputting a dataset of over 5000 point clouds. Large-scale models and VLN deployment are being integrated.

[2025 - now] Changpao Respiration (Entrepreneurship Project)

This student entrepreneurship project is funded by the base. We conducting hospital user interviews and competitive analysis of existing medical-grade respiratory devices, including tearing apart several units. The prototype development involves a unique "Dual Wind System," consisting of a "Barrier Wind" and "Circulating Wind" with dedicated mode construction. Wind tunnel test data is collected for control framework and structural design across multiple generations of prototypes, along with dual wind system fluid simulation. 10w fund secured, 2w school fund; additional 50w fund expected. Anticipating 3+ patents and papers.
